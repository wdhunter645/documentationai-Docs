## Overview

Bill Hunter helps you organize and manage all your project documentation in one centralized space. Create structured docs, collaborate with your team, and keep everything up-to-date effortlessly. With intuitive tools and customizable options, you streamline your documentation workflow from start to finish.

Use Bill Hunter to build knowledge bases, API references, user guides, and more. Access your docs via `https://docs.example.com` and integrate seamlessly with your existing tools.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create your free account. No credit card required.
  </Step>
  <Step title="Create Your First Doc" icon="file-plus">
    Click "New Document" and start writing in Markdown. Use this example:
    
````markdown
# My First Doc
Welcome to Bill Hunter!

## Features
- Easy editing
- Live preview
```
````
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Hit "Publish" to generate a live URL like `https://docs.example.com/my-first-doc`. Share with your team.
  </Step>
</Steps>
