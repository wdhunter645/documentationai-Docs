## Overview

Creating projects helps organize all your documentation in Bill Hunter.

## Steps

1. Go to your Bill Hunter dashboard.
2. Click “New Project” in the sidebar.
3. Enter a project name and description.
4. Click “Create”.

Your project is now ready. Start adding documentation and organize as needed.
